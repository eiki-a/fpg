松本研FPG勉強会のプログラム

11~20
いずれも zcat tweets.txt.gz | python 01X.pyで動作

11:「拡散希望」という文字列を含むツイートを標準出力
12:「なう」で終わるツイートを標準出力
13:RT先へのコメント（＝「RT 」という文字列より前の部分）を標準出力
14:ツイート中に含まれるユーザ名を標準出力
15:ツイート中に含まれるユーザ名を、そのユーザーのページへのリンクに置換したものを標準出力

16:漢字と大英字をグループに分け、タブ区切りしたものをtest.txtに出力
17:名前と接尾語（くん、ちゃん、さん）をグループに分け、タブ区切りしたものをtest.txtに出力
18:住所の形「（宮崎県）仙台市〜区〜（町村など、2から6文字）（番地：3から8文字もしくは番地＋アパート名＋部屋番＋「号」：任意の文字数）」
をtest.txtに出力
19:郵便番号（電話番号と区別するため、直前の文字が数字またはハイフンでないXXX-XXXX）と都道府県名（県名以外は個別一致、XX道などは特にミスマッチしやすいため）と市町村名（7文字以上の市町村名は存在しないので弾く）をグループに分け、ツイートがこれら3要素全てを含んでいれば取ってきて指定の形式にまとめtest.txtに出力
20:簡易顔文字が"("から始まることが多い（")"で終わるとは限らないことと文字数に着目して得た顔文字っぽいものをtest.txtに出力
